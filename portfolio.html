I have created a fictional "personal" website/blog, and my fictional character is a Web Developer. What is web development? Web Development usually refers to activities related to website functionalities development. There are three fundamentals of components that are conducted between the user and the site. Client (Frontend): Local compueter / Device & Browser - The client has to do with make sure that a website functions exactly the same way on all browsers.Server (Backend): Site code generation / Database handling - The server is the remote computer that is being run "on the other side" and is responsible for site code generation and database handling. Database: Information - The Database is the infomation that is generated or used within the website. For example, all acount information of a logged-in user is being stored in the database. Once the three fundamental components have been identified, you will clearly understand where web development ties into the entire process.
